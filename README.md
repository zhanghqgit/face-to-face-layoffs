# 你必须知道的裁员相关知识

鉴于这两天一直在发酵的网易员工被裁事件，必须懂得一些法律知识才能从容应对裁员风波。但在现实中，很多人对这些知识很淡薄，建立这个仓库的目的就是来弥补，可以从容的面对，有应对措施，怎么来维护自己的权益，而不是默默无闻，被动接受！

我也是在网上查找的一些资料，会有一些遗漏的地方，欢迎大家提交PR补充。

主要从以下几个方面介绍：

1. 心态，从容面对，无需示弱。
2. 收集证据
3. 法律知识
